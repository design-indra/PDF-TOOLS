Apa itu PDFTools?

PDFTools adalah kumpulan tools PDF gratis yang bisa digunakan langsung dari browser. Merge, split, compress, rotate PDF, dan konversi gambar ke PDF tanpa perlu install aplikasi.

5 Tools dalam 1 Website

  • 1

    Merge PDF: Gabungkan beberapa file PDF menjadi satu file yang rapi.

  • 2

    Split PDF: Ambil halaman tertentu dari PDF menjadi file baru.

  • 3

    Compress PDF: Kurangi ukuran file PDF agar mudah dikirim via WhatsApp atau email.

  • 4

    Rotate PDF: Putar halaman PDF yang terbalik atau miring.

  • 5

    Img to PDF: Konversi foto/gambar menjadi file PDF.

Fitur Utama:

  • 5 tools PDF dalam 1 halaman
  • Tidak perlu install aplikasi
  • File tidak disimpan di server
  • Gratis tanpa batas
  • Tidak perlu login/daftar
